# 518. Coin Change II

> [LeetCode 链接](https://leetcode.com/problems/coin-change-ii/)

给定不同面额的硬币和一个总金额。写出函数来计算可以凑成总金额的硬币**组合数**。假设每一种面额的硬币有**无限个**。

**示例 1**：
```
输入: amount = 5, coins = [1, 2, 5]
输出: 4
解释: 有四种方式可以凑成总金额:
5 = 5
5 = 2 + 2 + 1
5 = 2 + 1 + 1 + 1
5 = 1 + 1 + 1 + 1 + 1
```

**示例 2**：
```
输入: amount = 3, coins = [2]
输出: 0
```

**示例 3**：
```
输入: amount = 10, coins = [10]
输出: 1
```

## 题目分析

本题是典型的**完全背包**问题（每种硬币无限个），求的是**装满背包的组合数**。

**关键点**：
- 题目求的是**组合数**（不强调顺序），不是排列数
- `5 = 2 + 2 + 1` 和 `5 = 2 + 1 + 2` 算同一种组合
- 这一点会直接影响**遍历顺序**

与 494. 目标和 的区别：
- 494 是 **01 背包**（每种物品只有 1 个）
- 本题是**完全背包**（每种物品无限个）

---

## 二维 DP 解法

### 1. 确定 dp 数组及下标含义

`dp[i][j]`：使用下标为 `[0, i]` 的 `coins` 能够凑满容量为 `j` 的背包，有 `dp[i][j]` 种**组合方法**。

### 2. 确定递推公式

回顾几个递推公式的演化：

| 类型 | 递推公式 |
|---|---|
| 01 背包（最大价值） | `dp[i][j] = max(dp[i-1][j], dp[i-1][j-weight[i]] + value[i])` |
| 完全背包（最大价值） | `dp[i][j] = max(dp[i-1][j], dp[i][j-weight[i]] + value[i])` |
| 01 背包（组合数，494 目标和） | `dp[i][j] = dp[i-1][j] + dp[i-1][j-nums[i]]` |
| **完全背包（组合数，本题）** | **`dp[i][j] = dp[i-1][j] + dp[i][j-coins[i]]`** |

**本题递推公式**：

```
dp[i][j] = dp[i-1][j] + dp[i][j-coins[i]]
```

含义拆解：
- `dp[i-1][j]`：**不使用** `coins[i]` 的方案数（看上一行）
- `dp[i][j-coins[i]]`：**使用至少一枚** `coins[i]` 的方案数（看同行左边）
  - 注意是 `dp[i][...]` 不是 `dp[i-1][...]`，因为完全背包同一硬币可以继续使用

### 3. dp 数组初始化

**最左列 `dp[i][0]`**：背包容量为 0，唯一方法是什么都不装 → 全部初始化为 `1`

**最上行 `dp[0][j]`**：只用 `coins[0]` 凑金额 `j`

```java
for (int j = 0; j <= bagSize; j++) {
    if (j % coins[0] == 0) dp[0][j] = 1;
}
```

或者用递推形式（与主公式同构）：

```java
for (int j = coins[0]; j <= amount; j++) {
    dp[0][j] += dp[0][j - coins[0]];
}
```

**两种写法等价**：只有当 `j` 是 `coins[0]` 的倍数时方案数才是 1，否则为 0。

### 4. 遍历顺序

二维 DP 中，先遍历物品或先遍历背包都可以，因为不会影响递推公式所需的数值。

### 5. dp 数组举例

以 `amount = 5`，`coins = [2, 3, 5]` 为例：

```
        j=0  j=1  j=2  j=3  j=4  j=5
coin=2:  1    0    1    0    1    0
coin=3:  1    0    1    1    1    1
coin=5:  1    0    1    1    1    2
```

### 代码实现

#### Java

```java
class Solution {
    public int change(int amount, int[] coins) {
        int[][] dp = new int[coins.length][amount + 1];

        // 初始化最左列：装满容量 0，1 种方法（什么都不装）
        for (int i = 0; i < coins.length; i++) {
            dp[i][0] = 1;
        }
        // 初始化最上行：只用 coins[0] 凑金额 j
        for (int j = coins[0]; j <= amount; j++) {
            dp[0][j] += dp[0][j - coins[0]];
        }

        // 填表
        for (int i = 1; i < coins.length; i++) {
            for (int j = 1; j <= amount; j++) {
                if (j < coins[i]) {
                    dp[i][j] = dp[i - 1][j];
                } else {
                    dp[i][j] = dp[i - 1][j] + dp[i][j - coins[i]];
                }
            }
        }

        return dp[coins.length - 1][amount];
    }
}
```

## 一维 DP 解法（推荐）

### 1. dp 数组含义

`dp[j]`：凑成总金额 `j` 的货币**组合数**为 `dp[j]`

### 2. 递推公式

由二维压缩而来：

```
dp[j] += dp[j - coins[i]]
```

### 3. 初始化

`dp[0] = 1`（凑 0 元只有 1 种方法：什么都不选）

### 4. 遍历顺序（重点）

> **求组合数 vs 求排列数，遍历顺序不同！**

**外层物品，内层背包 → 组合数**（本题）：

```java
for (int i = 0; i < coins.size(); i++) {        // 外层：物品
    for (int j = coins[i]; j <= amount; j++) {  // 内层：背包
        dp[j] += dp[j - coins[i]];
    }
}
```

假设 `coins = [1, 5]`：先把 1 全部加入计算，再把 5 加入。只会产生 `{1, 5}`，**不会**产生 `{5, 1}`。

**外层背包，内层物品 → 排列数**：

```java
for (int j = 0; j <= amount; j++) {              // 外层：背包
    for (int i = 0; i < coins.size(); i++) {     // 内层：物品
        if (j - coins[i] >= 0) dp[j] += dp[j - coins[i]];
    }
}
```

每个背包容量都会被 1 和 5 各算一次，`{1, 5}` 和 `{5, 1}` 都被算进去。

### 关键决策

| 求什么 | 外层 | 内层 |
|---|---|---|
| **组合数** | 物品 | 背包 |
| **排列数** | 背包 | 物品 |

### 代码实现

#### Java

```java
class Solution {
    public int change(int amount, int[] coins) {
        int[] dp = new int[amount + 1];
        dp[0] = 1; // 凑 0 元只有一种方式（什么都不装）

        for (int i = 0; i < coins.length; i++) {        // 遍历物品
            for (int j = coins[i]; j <= amount; j++) {  // 遍历背包
                dp[j] += dp[j - coins[i]];
            }
        }

        return dp[amount];
    }
}
```

#### Python

```python
class Solution:
    def change(self, amount: int, coins: List[int]) -> int:
        dp = [0] * (amount + 1)
        dp[0] = 1
        # 遍历物品
        for i in range(len(coins)):
            # 遍历背包
            for j in range(coins[i], amount + 1):
                dp[j] += dp[j - coins[i]]
        return dp[amount]
```

**复杂度分析**：
- 时间复杂度：`O(m × n)`，其中 `m` 是 `amount`，`n` 是 `coins.length`
- 空间复杂度：`O(m)`

---

## 总结

1. **题目类型**：完全背包 + 求组合数
2. **递推公式**：`dp[j] += dp[j - coins[i]]`（一维）或 `dp[i][j] = dp[i-1][j] + dp[i][j-coins[i]]`（二维）
3. **初始化**：`dp[0] = 1`，凑 0 元有 1 种方法
4. **遍历顺序（关键）**：
   - **组合数** → 外层物品，内层背包
   - **排列数** → 外层背包，内层物品
5. **完全背包 vs 01 背包的区别**：递推公式中 `dp[i][j-coins[i]]` vs `dp[i-1][j-coins[i]]`，前者允许同一物品被重复使用
