# 222. 完全二叉树的节点个数 (Count Complete Tree Nodes)

> [LeetCode 222](https://leetcode.com/problems/count-complete-tree-nodes/)

## 题目理解

给出一个**完全二叉树**，求节点个数。要求时间复杂度优于 O(n)。

**完全二叉树性质**：除了最后一层，每层都是满的；最后一层节点靠左排列。

## 思路一：普通二叉树递归（O(n)）

后序遍历，左子树节点数 + 右子树节点数 + 1（自己）。

```java
class Solution {
    public int countNodes(TreeNode root) {
        if (root == null) return 0;
        return countNodes(root.left) + countNodes(root.right) + 1;
    }
}
```

- **时间复杂度**：O(n)，每个节点都访问一次
- **空间复杂度**：O(log n)，递归栈深度

## 思路二：利用完全二叉树性质（O(log²n)）

### 核心思路

完全二叉树的任意子树，要么是**满二叉树**，要么不是：

- **满二叉树**：直接用公式 `2^h - 1` 计算节点数
- **不是满二叉树**：递归左右子树，继续判断

### 如何判断满二叉树？

从当前节点一路往**最左**走，记深度 `leftDepth`；一路往**最右**走，记深度 `rightDepth`。

- `leftDepth == rightDepth` → 满二叉树，用公式
- `leftDepth != rightDepth` → 不是满的，递归两边

**为什么这个判断成立？** 因为完全二叉树的"缺口"只可能在最后一层的右侧。如果最左路径和最右路径一样长，中间不可能有空洞，一定是满的。

### 关于 `(2 << leftDepth) - 1`

`<<` 是位左移运算符，左移一位 = 乘以 2：

| 表达式 | 结果 | 等价于 |
|---|---|---|
| `2 << 0` | 2 | 2¹ |
| `2 << 1` | 4 | 2² |
| `2 << 2` | 8 | 2³ |

规律：**`2 << n` = 2^(n+1)**

代码中 `leftDepth` 从 0 开始（root 的子节点开始计数），实际层数 = `leftDepth + 1`，所以 `(2 << leftDepth) - 1` = 2^(leftDepth+1) - 1，正好是满二叉树节点数。

### 递归过程举例

以 `[1, 2, 3, 4, 5, 6]` 为例：

1. **root=1**：leftDepth=2（1→2→4），rightDepth=1（1→3→6）。2≠1，不是满树，递归两边
2. **root=2（左子树）**：leftDepth=1（2→4），rightDepth=1（2→5）。1==1，满树！节点数 = `(2<<1)-1 = 3`
3. **root=3（右子树）**：leftDepth=1（3→6），rightDepth=0（3 没有右子树）。1≠0，继续递归
4. **root=6（左子）**：leftDepth=0，rightDepth=0，满树！= 1 节点
5. **root=null（右子）**：返回 0
6. 节点 3 的子树 = 1 + 0 + 1 = 2
7. **总计** = 3 + 2 + 1 = **6**

## 最终代码

```java
class Solution {
    /**
     * 针对完全二叉树的解法
     * 满二叉树的结点数为：2^depth - 1
     */
    public int countNodes(TreeNode root) {
        if (root == null) return 0;
        TreeNode left = root.left;
        TreeNode right = root.right;
        int leftDepth = 0, rightDepth = 0; // 初始为0，为了下面求指数方便
        while (left != null) {  // 求左子树深度（一路往左）
            left = left.left;
            leftDepth++;
        }
        while (right != null) { // 求右子树深度（一路往右）
            right = right.right;
            rightDepth++;
        }
        if (leftDepth == rightDepth) {
            return (2 << leftDepth) - 1; // 满二叉树，用公式
        }
        return countNodes(root.left) + countNodes(root.right) + 1; // 不满，递归两边
    }
}
```

## 复杂度分析

- **时间复杂度**：O(log²n)。递归深度 O(log n) 层，每层求深度 O(log n)
- **空间复杂度**：O(log n)，递归栈深度

## 总结

1. **普通做法**（O(n)）：后序遍历统计所有节点，不利用完全二叉树性质
2. **优化做法**（O(log²n)）：利用完全二叉树性质，判断子树是否为满二叉树，满的用公式，不满的递归
3. 判断满二叉树的方法：**最左深度 == 最右深度**，这在完全二叉树前提下成立
4. `(2 << leftDepth) - 1` 是 `2^(leftDepth+1) - 1` 的位运算写法，因为 `leftDepth` 从 0 开始
5. 递归下去一定会碰到满二叉树（最差情况是单个叶子节点），所以一定能用公式收敛
