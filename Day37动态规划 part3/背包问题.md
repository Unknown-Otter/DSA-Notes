
# 01 背包理论基础

> [卡码网第46题 - 携带研究材料](https://kamacoder.com/problempage.php?pid=1046)

## 一、问题定义

有 `n` 件物品和一个最多能背重量为 `W` 的背包。第 `i` 件物品的重量是 `weight[i]`，价值是 `value[i]`。**每件物品只能用一次**，求将哪些物品装入背包能获得最大价值。

**关键特征**：每件物品只能用一次（这是 01 背包区别于完全背包的根本）。

### 贯穿全文的样例

背包容量 `W = 4`，物品如下：

| 物品   | 重量 | 价值 |
| ------ | ---- | ---- |
| 物品 0 | 1    | 15   |
| 物品 1 | 3    | 20   |
| 物品 2 | 4    | 30   |

**最优解**：装物品 0 + 物品 1，重量 1+3=4，价值 15+20=**35**

## 二、二维 dp 数组解法

### 1. dp 数组定义

`dp[i][j]` 表示：从下标为 `[0, i]` 的物品里任意取，放进容量为 `j` 的背包，能获得的**最大价值**。

- `i` 表示物品（行）
- `j` 表示背包容量（列）

### 2. 递推公式

对每个 `dp[i][j]`，只有两种选择：

- **不放物品 i**：`dp[i-1][j]`（沿用上一行同列的值）
- **放物品 i**：`dp[i-1][j-weight[i]] + value[i]`（腾出空间后加上物品 i 的价值）

```
dp[i][j] = max(dp[i-1][j], dp[i-1][j-weight[i]] + value[i])
```

### 3. 初始化

- **第一列** `dp[i][0] = 0`：背包容量为 0，啥也装不下
- **第一行** `dp[0][j]`：只考虑物品 0 时，容量够（`j >= weight[0]`）就装，值为 `value[0]`；否则为 0
- **其他位置** 初始化为 0 即可，会被覆盖

### 4. 遍历顺序

两个 for 循环：**先物品后容量** 或 **先容量后物品** 都行（二维下两种顺序均可），但**先物品后容量更直观**。

### 5. 二维 Java 代码

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int n = scanner.nextInt();              // 物品数量
        int bagweight = scanner.nextInt();      // 背包容量

        int[] weight = new int[n];
        int[] value = new int[n];

        for (int i = 0; i < n; ++i) weight[i] = scanner.nextInt();
        for (int j = 0; j < n; ++j) value[j] = scanner.nextInt();

        int[][] dp = new int[n][bagweight + 1];

        // 初始化第一行
        for (int j = weight[0]; j <= bagweight; j++) {
            dp[0][j] = value[0];
        }

        for (int i = 1; i < n; i++) {
            for (int j = 0; j <= bagweight; j++) {
                if (j < weight[i]) {
                    dp[i][j] = dp[i - 1][j];
                } else {
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i - 1][j - weight[i]] + value[i]);
                }
            }
        }

        System.out.println(dp[n - 1][bagweight]);
    }
}
```

### 6. 二维 Python 代码

```python
n, bagweight = map(int, input().split())
weight = list(map(int, input().split()))
value = list(map(int, input().split()))

# 创建 n 行 bagweight+1 列的二维列表,全部为 0
dp = [[0] * (bagweight + 1) for _ in range(n)]

# 初始化第一行
for j in range(weight[0], bagweight + 1):
    dp[0][j] = value[0]

for i in range(1, n):
    for j in range(bagweight + 1):
        if j < weight[i]:
            dp[i][j] = dp[i - 1][j]
        else:
            dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - weight[i]] + value[i])

print(dp[n - 1][bagweight])
```

### Python 语法点（针对新手）

| 语法                                | 含义                                                      |
| ----------------------------------- | --------------------------------------------------------- |
| `map(int, input().split())`         | 读一行,按空格切开,每个元素转 int                          |
| `list(map(...))`                    | 把 map 迭代器实化为列表,可下标访问                        |
| `[0] * size`                        | 创建长度为 size、全为 0 的列表                            |
| `[[0]*c for _ in range(n)]`         | 创建二维列表的**正确**写法（不要用 `[[0]*c]*n`,会引用陷阱） |
| `range(a, b)`                       | 生成 `a, a+1, ..., b-1`(**左闭右开**)                     |
| `_`                                 | 约定俗成的"用不到的变量"占位符                            |
| 缩进                                | 代替花括号,通常 4 个空格                                  |

## 三、一维 dp 数组（滚动数组）解法

### 1. 为什么可以压缩成一维

观察二维递推式：`dp[i][j] = max(dp[i-1][j], dp[i-1][j-weight[i]] + value[i])`

只用到了**上一行** `dp[i-1]` 的数据,更早的行用不到。所以可以用一行覆盖更新,即"滚动数组"。

### 2. dp 数组定义

`dp[j]` 表示：容量为 `j` 的背包,所背物品的最大价值。

### 3. 递推公式

把二维公式去掉 `i` 维度：

```
dp[j] = max(dp[j], dp[j - weight[i]] + value[i])
```

- 等号左边的 `dp[j]` 是**今天要更新的新值**
- 等号右边的 `dp[j]` 和 `dp[j-weight[i]]` 是**昨天留下的旧值**

### 4. 初始化

`dp` 数组**全部初始化为 0** 即可（前提:物品价值都是非负数）。

### 5. 遍历顺序（重点!）

#### ⭐ 必须倒序遍历容量

**原因**：防止同一物品被重复装入。

**举例**：物品 0 重量 1,价值 15。初始 `dp = [0,0,0,0,0]`

❌ **正序遍历**（错误）：

```
j=1: dp[1] = max(0, dp[0]+15) = 15
j=2: dp[2] = max(0, dp[1]+15) = 30  ← 物品0被装了两次!
```

✅ **倒序遍历**（正确）：

```
j=4: dp[4] = max(0, dp[3]+15) = 15
j=3: dp[3] = max(0, dp[2]+15) = 15
j=2: dp[2] = max(0, dp[1]+15) = 15
j=1: dp[1] = max(0, dp[0]+15) = 15
```

倒序时,`dp[j-weight[i]]` 读到的是**昨天的旧值**(还没更新),所以物品只会装一次。

#### ⭐ 外层必须是物品,内层是容量

如果颠倒（外层容量、内层物品）,每个 `dp[j]` 在内层循环中被反复更新,等价于**每个容量的背包只装了一件物品**,逻辑全错。

**口诀**：**外层物品,内层容量倒序**

### 6. 一维 Java 代码

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int M = scanner.nextInt();   // 物品数量
        int N = scanner.nextInt();   // 背包容量

        int[] costs = new int[M];
        int[] values = new int[M];

        for (int i = 0; i < M; i++) costs[i] = scanner.nextInt();
        for (int j = 0; j < M; j++) values[j] = scanner.nextInt();

        int[] dp = new int[N + 1];

        for (int i = 0; i < M; i++) {                       // 外层:物品
            for (int j = N; j >= costs[i]; j--) {           // 内层:容量倒序
                dp[j] = Math.max(dp[j], dp[j - costs[i]] + values[i]);
            }
        }

        System.out.println(dp[N]);
        scanner.close();
    }
}
```

### 7. 一维 Python 代码

```python
n, bagweight = map(int, input().split())
weight = list(map(int, input().split()))
value = list(map(int, input().split()))

dp = [0] * (bagweight + 1)

for i in range(n):                                      # 外层:物品
    for j in range(bagweight, weight[i] - 1, -1):       # 内层:容量倒序
        dp[j] = max(dp[j], dp[j - weight[i]] + value[i])

print(dp[bagweight])
```

### Python 倒序 range 详解

```python
for j in range(bagweight, weight[i] - 1, -1):
```

`range(start, stop, step)` 三个参数：

- `start = bagweight` —— 起点(包含)
- `stop = weight[i] - 1` —— 终点(**不包含**)
- `step = -1` —— 每次减 1(倒着走)

所以 `j` 依次取：`bagweight, bagweight-1, ..., weight[i]`

为什么停在 `weight[i]`？再小就装不下物品 i 了,没必要更新。

## 四、手动推导样例（一维 dp）

样例：`n=3, bagweight=4, weight=[1,3,4], value=[15,20,30]`

| 阶段              | dp[0] | dp[1] | dp[2] | dp[3] | dp[4] |
| ----------------- | ----- | ----- | ----- | ----- | ----- |
| 初始              | 0     | 0     | 0     | 0     | 0     |
| 处理完物品 0      | 0     | 15    | 15    | 15    | 15    |
| 处理完物品 1      | 0     | 15    | 15    | 20    | **35** |
| 处理完物品 2      | 0     | 15    | 15    | 20    | 35    |

**最终答案**：`dp[4] = 35` ✅

## 五、一维 vs 二维对比

| 对比点         | 二维 dp                | 一维 dp           |
| -------------- | ---------------------- | ----------------- |
| 数组定义       | `dp[i][j]`             | `dp[j]`           |
| 空间复杂度     | O(n × W)               | O(W)              |
| 容量遍历方向   | 正序、倒序都行         | **必须倒序**      |
| 循环顺序       | 物品/容量都可在外层    | **物品必须在外层** |
| 初始化         | 第一行需单独处理       | 全 0 即可         |
| 代码简洁度     | 较繁琐                 | 简洁              |

> 推荐使用一维 dp,代码更短、空间更优,后续背包变种题也都用一维。

## 六、灵魂三问（自检掌握程度）

1. **为什么一维 dp 必须倒序遍历容量？**
   → 防止物品被重复装入。倒序时读到的左边格子是旧值,保证每件物品只装一次。

2. **为什么外层必须是物品？**
   → 容量在外层会导致每个 `dp[j]` 只装入一件物品,逻辑错误。

3. **为什么二维 dp 不需要倒序？**
   → 二维 dp 用的是 `dp[i-1][...]`(上一行),与当前行的更新不冲突。

## 七、最终总结

1. **01 背包核心**：每件物品只能用一次,二选一(装/不装)取最大值
2. **递推公式**：
   - 二维：`dp[i][j] = max(dp[i-1][j], dp[i-1][j-weight[i]] + value[i])`
   - 一维：`dp[j] = max(dp[j], dp[j-weight[i]] + value[i])`
3. **一维写法口诀**：**外层物品,内层容量倒序**
4. **力扣应用**：分割等和子集、目标和、最后一块石头的重量 II、零钱兑换 II 等都是 01 背包/完全背包的变种
