
# 530. Minimum Absolute Difference in BST

> [LeetCode 530](https://leetcode.com/problems/minimum-absolute-difference-in-bst/)

## 核心思路

**BST + 中序遍历 = 天然升序序列**，这是 BST 最核心的性质之一。既然中序遍历结果是递增的，那最小绝对差一定出现在**相邻两个节点**之间。所以只需要中序遍历时用 `pre` 记录上一个节点，边遍历边比较即可，不需要真的建数组。

## 解法一：递归（中序遍历）

```java
class Solution {
    TreeNode pre; // 记录上一个遍历的结点
    int result = Integer.MAX_VALUE;

    public int getMinimumDifference(TreeNode root) {
        if (root == null)
            return 0;
        traversal(root);
        return result;
    }

    public void traversal(TreeNode root) {
        if (root == null)
            return;
        // 左
        traversal(root.left);
        // 中
        if (pre != null) {
            result = Math.min(result, root.val - pre.val);
        }
        pre = root;
        // 右
        traversal(root.right);
    }
}
```

## 解法二：迭代法（中序遍历）

```java
class Solution {
    TreeNode pre;
    Stack<TreeNode> stack;
    public int getMinimumDifference(TreeNode root) {
        if (root == null) return 0;
        stack = new Stack<>();
        TreeNode cur = root;
        int result = Integer.MAX_VALUE;
        while (cur != null || !stack.isEmpty()) {
            if (cur != null) {
                stack.push(cur); // 将访问的节点放进栈
                cur = cur.left; // 左
            } else {
                cur = stack.pop();
                if (pre != null) { // 中
                    result = Math.min(result, cur.val - pre.val);
                }
                pre = cur;
                cur = cur.right; // 右
            }
        }
        return result;
    }
}
```

## 解法三：统一迭代法（中序遍历 + null 标记）

```java
class Solution {
    public int getMinimumDifference(TreeNode root) {
        Stack<TreeNode> stack = new Stack<>();
        TreeNode pre = null;
        int result = Integer.MAX_VALUE;

        if (root != null)
            stack.add(root);

        // 中序遍历（左中右），由于栈先入后出，反序（右中左）
        while (!stack.isEmpty()) {
            TreeNode curr = stack.peek();
            if (curr != null) {
                stack.pop();
                // 右
                if (curr.right != null)
                    stack.add(curr.right);
                // 中（先用null标记）
                stack.add(curr);
                stack.add(null);
                // 左
                if (curr.left != null)
                    stack.add(curr.left);
            } else { // 中（遇到null再处理）
                stack.pop();
                TreeNode temp = stack.pop();
                if (pre != null)
                    result = Math.min(result, temp.val - pre.val);
                pre = temp;
            }
        }
        return result;
    }
}
```

## 踩坑记录

- **pre 要记录的是 TreeNode 而不是 int**：用 `TreeNode pre` 可以通过 `pre != null` 自然判断是否是第一个节点；如果用 `int prev = -1` 之类的值，当节点值本身就是 -1 时会出 bug
- **中序遍历的有序性是 BST 专属**：普通二叉树中序遍历不保证有序，这题能这么做完全依赖 BST 性质

## 复杂度

- **时间**：O(n)，每个节点访问一次
- **空间**：O(h)，h 为树高，递归栈 / 显式栈的开销

## 总结

1. BST 中序遍历 = 升序序列，最小差一定在相邻元素间
2. 用 `pre` 指针边遍历边比较，无需额外数组
3. 三种写法（递归、经典迭代、统一迭代）本质相同，都是中序遍历 + 双指针比较
