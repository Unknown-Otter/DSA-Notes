# LC 383. Ransom Note

> https://leetcode.com/problems/ransom-note/

## 题目描述

给定两个字符串 `ransomNote` 和 `magazine`，判断 `ransomNote` 中的每个字母是否都能从 `magazine` 中找到（每个字母只能使用一次）。

---

## 解题思路

### Step 1：统计 magazine 的字母频次

用 `HashMap<Character, Integer>` 记录 `magazine` 中每个字母出现的次数。

```java
for (char letter : magazine.toCharArray()) {
    map.put(letter, map.getOrDefault(letter, 0) + 1);
}
```

### Step 2：遍历 ransomNote，逐字母核销

对 `ransomNote` 中的每个字母，检查 map 中该字母的剩余数量：
- 若数量 ≤ 0（不存在或已用完）→ 直接返回 `false`
- 否则将该字母的计数 -1（消耗掉一次）

```java
for (int i = 0; i < ransomNote.length(); i++) {
    char c = ransomNote.charAt(i);
    if (map.getOrDefault(c, 0) <= 0) {
        return false;
    }
    map.put(c, map.get(c) - 1);
}
```

### Step 3：返回结果

走到循环结束说明所有字母都能被满足，返回 `true`。

---

## 完整代码

```java
class Solution {
    public boolean canConstruct(String ransomNote, String magazine) {
        Map<Character, Integer> map = new HashMap<>();

        for (char letter : magazine.toCharArray()) {
            map.put(letter, map.getOrDefault(letter, 0) + 1);
        }

        for (int i = 0; i < ransomNote.length(); i++) {
            char c = ransomNote.charAt(i);
            if (map.getOrDefault(c, 0) <= 0) {
                return false;
            }
            map.put(c, map.get(c) - 1);
        }

        return true;
    }
}
```

### Bug 2：循环类型不匹配

```java
// ❌ 错误：用 int 接收 char（虽然能编译，但语义不清晰）
for(int letter : magazine)

// ✅ 正确
for(char letter : magazine.toCharArray())
```

**教训**：`String` 不能直接 for-each，需要 `.toCharArray()`；接收字符用 `char`，不要用 `int`。

---

### Bug 3：逻辑漏洞——没有检查 value 是否已经为 0（最关键）

```java
// ❌ 错误：只检查 key 存不存在，没检查剩余数量
if (map.containsKey(ransomNote.charAt(i))) {
    map.put(..., count - 1);  // count 可能减成负数，静默错误！
}
```

**原因分析**：
- `HashMap` 的 value 是 `Integer`，可以存负数，**数据结构本身不会报错**
- 但负数在这题里意味着"magazine 里这个字母不够用了"
- 这类 bug **没有运行时异常**，只会静默地给出错误答案
- 需要专门构造"字母数量刚好不够"的 case 才能暴露（如 `ransomNote = "aa"`, `magazine = "a"`）

```java
// ✅ 正确：检查数量是否充足
if (map.getOrDefault(c, 0) <= 0) {
    return false;
}
map.put(c, map.get(c) - 1);
```

// ✅ 正确

## 总结

1. **用 HashMap 统计 magazine 字频**，key = 字母，value = 出现次数
2. **遍历 ransomNote**，每次检查剩余数量是否 > 0，再执行 -1
3. **检查数量用 `getOrDefault(c, 0) <= 0`**，不要只用 `containsKey`
4. HashMap 不会对负数 value 报错，**逻辑漏洞只能靠边界 case 测试发现**
