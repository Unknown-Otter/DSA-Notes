# 102. 二叉树的层序遍历

> [LeetCode 链接](https://leetcode.cn/problems/binary-tree-level-order-traversal/)

给你一个二叉树，请你返回其按 **层序遍历** 得到的节点值（即逐层地，从左到右访问所有节点）。

---

## 一、核心思路

层序遍历就是从左到右**一层一层**地去遍历二叉树，本质上就是图论中的 **广度优先遍历（BFS）**，只不过应用在二叉树上。

**关键对比**：

- **队列（先进先出）** → 适合层序遍历（BFS），符合一层一层处理的逻辑
- **栈（先进后出）** → 适合模拟深度优先遍历（DFS）的递归逻辑

因此，层序遍历需要借助 **队列** 这一辅助数据结构来实现。

---

## 二、迭代法（BFS，推荐模板）

### 实现要点

1. 根节点先入队（注意判空）
2. 每一层开始时，**固定记录当前队列大小 `size`**（不能直接用 `que.size()`，因为它在循环中会变化）
3. 一次性弹出当前层的所有节点，并把它们的左右子节点加入队列
4. 每处理完一层，将该层结果加入最终结果集

> **这份代码可以作为二叉树层序遍历的模板，"打十个就靠它了"。**

### Java 代码

```java
class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> resList = new ArrayList<>();
        if (root == null) return resList;

        Queue<TreeNode> que = new LinkedList<>();
        que.offer(root);

        while (!que.isEmpty()) {
            List<Integer> itemList = new ArrayList<>();
            int len = que.size();  // 固定当前层大小

            while (len > 0) {
                TreeNode tmpNode = que.poll();
                itemList.add(tmpNode.val);

                if (tmpNode.left != null) que.offer(tmpNode.left);
                if (tmpNode.right != null) que.offer(tmpNode.right);
                len--;
            }

            resList.add(itemList);
        }
        return resList;
    }
}
```

## 三、递归法（DFS 模拟层序）

### 实现要点

通过传入 **当前深度 `deep`** 作为参数，利用 `resList` 的索引来界定层级：

- 当 `resList.size() < deep` 时，说明这是新的一层，需要新建一个子列表
- 用 `deep - 1` 作为索引，把当前节点值放到对应层

### Java 代码

```java
class Solution {
    public List<List<Integer>> resList = new ArrayList<>();

    public List<List<Integer>> levelOrder(TreeNode root) {
        checkFun01(root, 0);
        return resList;
    }

    public void checkFun01(TreeNode node, Integer deep) {
        if (node == null) return;
        deep++;

        if (resList.size() < deep) {
            // 当层级增加时，list 的 item 也增加，利用 list 的索引值进行层级界定
            List<Integer> item = new ArrayList<>();
            resList.add(item);
        }
        resList.get(deep - 1).add(node.val);

        checkFun01(node.left, deep);
        checkFun01(node.right, deep);
    }
}
```

---

## 四、总结

1. **层序遍历 = BFS**，借助 **队列** 实现，是二叉树遍历的另一种重要方式
2. **迭代法模板** 的关键：在每层开始前用 `int len = que.size()` 锁定当前层节点数
3. **递归法** 通过传递 `depth` 参数，利用结果列表索引天然映射层级关系
4. 这套层序遍历模板可以解决一系列层序相关的衍生题目（如锯齿层序、右视图、平均值等）
