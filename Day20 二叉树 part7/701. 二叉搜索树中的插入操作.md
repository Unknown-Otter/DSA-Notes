# 701. 二叉搜索树中的插入操作

> [LeetCode 701](https://leetcode.cn/problems/insert-into-a-binary-search-tree/)

给定二叉搜索树（BST）的根节点和要插入树中的值，将值插入二叉搜索树。返回插入后二叉搜索树的根节点。输入数据保证新值和原始二叉搜索树中的任意节点值都不同。

**注意**：可能存在多种有效的插入方式，只要树在插入后仍保持为二叉搜索树即可。你可以返回任意有效的结果。

## 核心思路

这道题看似复杂（题目提示可以重构二叉搜索树），但其实**根本不需要重构**。

**最简单的策略：按照 BST 规则比大小选方向，遇到空节点就插入。** 新节点一定插在叶子位置，不需要移动任何已有节点。

## 递归法（有返回值）

### 递归三部曲

**1. 确定递归函数参数及返回值**

- 参数：根节点指针 + 要插入的值
- 返回值：`TreeNode` 类型，利用返回值完成新节点与父节点的赋值操作

**2. 确定终止条件**

- 遍历到 `null` 节点，说明找到了插入位置，创建新节点并返回

**3. 确定单层递归逻辑**

- BST 是有方向的搜索树，根据插入元素的值决定递归方向，**不需要遍历整棵树**
- `root.val > val` → 往左走
- `root.val < val` → 往右走
- **关键**：用 `root.left = insertIntoBST(root.left, val)` 接住返回值，下一层返回新节点，本层用 `root.left` 或 `root.right` 将其接住

```java
class Solution {
    public TreeNode insertIntoBST(TreeNode root, int val) {
        if (root == null) // 找到合适位置，创建节点直接返回
            return new TreeNode(val);

        if (root.val < val) {
            root.right = insertIntoBST(root.right, val); // 递归创建右子树
        } else if (root.val > val) {
            root.left = insertIntoBST(root.left, val); // 递归创建左子树
        }
        return root;
    }
}
```

### 为什么 `root.left = insertIntoBST(...)` 能正确工作？

- **触底时**（`root == null`）：返回 `new TreeNode(val)`，上一层通过 `root.left =` 或 `root.right =` 把新节点挂上
- **中间节点**：递归返回的就是原节点本身，`root.left = root.left` 等于没变，不会破坏原有结构
- **只有 null 那一层真正创建了新节点，其余层只是把原来的孩子重新赋值了一遍**

## 迭代法

迭代法需要额外记录**父节点**（`pre`），因为遍历到 `null` 时需要知道是谁的孩子。

```java
class Solution {
    public TreeNode insertIntoBST(TreeNode root, int val) {
        if (root == null) return new TreeNode(val);

        TreeNode newRoot = root;
        TreeNode pre = root;
        while (root != null) {
            pre = root;
            if (root.val > val) {
                root = root.left;
            } else if (root.val < val) {
                root = root.right;
            }
        }
        if (pre.val > val) {
            pre.left = new TreeNode(val);
        } else {
            pre.right = new TreeNode(val);
        }

        return newRoot;
    }
}
```

## 踩坑记录

1. **比较对象写错**：应该与参数 `val` 比较，而不是硬编码数字（如 `25`）
2. **递归结果没接回去**：写成 `TreeNode left = insertIntoBST(root.left, val)` 导致新节点创建了但没有挂到树上，必须写 `root.left = insertIntoBST(root.left, val)`
3. **不需要重构 BST**：题目说"可能存在多种有效的插入方式"容易吓退人，其实只要插到叶子位置就行

## 总结

1. BST 插入不需要重构树结构，只需找到空节点位置插入即可
2. 递归函数的返回值可以巧妙地完成父子节点的赋值操作（`root.left = insertIntoBST(...)`），这是一个通用技巧
3. 没有返回值的递归写法需要额外记录 parent 节点，迭代法同理——都比有返回值的写法更麻烦
4. 对于一棵固定的 BST 和给定值，插入位置并不唯一，但插到叶子节点是最简单的策略
5. 时间复杂度 O(n)，空间复杂度 O(n)（最坏情况退化为链表）；平均情况 O(log n)
