
# 654. Maximum Binary Tree

> [LeetCode 654](https://leetcode.com/problems/maximum-binary-tree/)

## 题意理解

给定一个**无重复元素**的整数数组 `nums`，按以下规则递归构建**最大二叉树**：

1. 数组中的**最大值**作为当前根节点
2. 最大值**左边的子数组**递归构建左子树
3. 最大值**右边的子数组**递归构建右子树

**关键点：左/右孩子不是由大小关系决定的（不是BST），而是由元素在数组中相对于最大值的位置决定的。**

## 递归思路

1. **终止条件**：数组为空，返回 `null`
2. **找最大值及其下标** `maxIndex`
3. **创建根节点** `new TreeNode(nums[maxIndex])`
4. **切割数组**：`[0, maxIndex)` 为左子数组，`(maxIndex, nums.length)` 为右子数组
5. **递归**构建左右子树

## 递归过程图解

```
[3, 2, 1, 6, 0, 5]
          ^max=6
左[3,2,1]        右[0,5]

[3, 2, 1]        [0, 5]
 ^max=3            ^max=5
左[] 右[2,1]     左[0] 右[]

     [2, 1]
      ^max=2
    左[] 右[1]

最终的树：
        6
       / \
      3   5
       \  /
        2 0
         \
          1
```

## 代码

```java
class Solution {
    public TreeNode constructMaximumBinaryTree(int[] nums) {
        if (nums.length == 0) return null;

        // 找最大值的同时记录下标
        int maxIndex = 0;
        for (int i = 1; i < nums.length; i++) {
            if (nums[i] > nums[maxIndex]) {
                maxIndex = i;
            }
        }

        TreeNode root = new TreeNode(nums[maxIndex]);

        // copyOfRange(原数组, 起始index(含), 结束index(不含))
        int[] leftNums = Arrays.copyOfRange(nums, 0, maxIndex);
        int[] rightNums = Arrays.copyOfRange(nums, maxIndex + 1, nums.length);

        root.left = constructMaximumBinaryTree(leftNums);
        root.right = constructMaximumBinaryTree(rightNums);

        return root;
    }
}
```

## 踩坑记录

1. **只记录 max 值不记录 index**：找最大值时要同时记录 `maxIndex`，否则无法知道从哪里切割数组。技巧：不存 `max` 值，只存 `maxIndex`，用 `nums[maxIndex]` 获取最大值
2. **左右孩子的判断依据**：不是按值的大小（不是BST），而是按**数组下标位置**——在最大值左边的元素进左子树，右边的进右子树
3. **`Arrays.copyOfRange` 的范围**：左半边是 `[0, maxIndex)`（不含 maxIndex），右半边是 `[maxIndex + 1, nums.length)`

## 复杂度分析

- **时间复杂度**：平均 O(n log n)，最坏 O(n²)（数组已排序时，每次最大值在边界，退化为链表）
- **空间复杂度**：O(n)（递归栈深度 + 每次创建新数组的开销）

## 总结

1. 本题是**构造二叉树**类型题，核心模式是「找分割点 → 切割数组 → 递归」
2. 左右孩子由**数组位置**决定，不是值的大小关系
3. 找最大值时用 **maxIndex** 一箭双雕：既能取值，又能切割
4. `Arrays.copyOfRange(nums, from, to)` 是 Java 中切割数组的常用工具，左闭右开
