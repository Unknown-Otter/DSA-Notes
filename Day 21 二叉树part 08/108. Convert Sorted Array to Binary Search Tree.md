# 108. Convert Sorted Array to Binary Search Tree

> [LeetCode 108](https://leetcode.com/problems/convert-sorted-array-to-binary-search-tree/)

## 思路

给定一个**升序排列**的整数数组，构造一棵**高度平衡**的二叉搜索树（BST）。

核心思想：**每次取区间中点作为根节点**，左半段递归构建左子树，右半段递归构建右子树。因为数组有序，中点分割天然满足 BST 性质；每次对半分也保证了高度平衡。

## 递归：左闭右开 [left, right)

```java
class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        return sortedArrayToBST(nums, 0, nums.length);
    }
    
    public TreeNode sortedArrayToBST(int[] nums, int left, int right) {
        if (left >= right) {
            return null;
        }
        if (right - left == 1) {
            return new TreeNode(nums[left]);
        }
        int mid = left + (right - left) / 2;
        TreeNode root = new TreeNode(nums[mid]);
        root.left = sortedArrayToBST(nums, left, mid);
        root.right = sortedArrayToBST(nums, mid + 1, right);
        return root;
    }
}
```

**关键点：**
- 初始调用传 `nums.length`（不是 `nums.length - 1`），因为右边界是开区间
- 终止条件是 `left >= right`（区间为空）
- 左子树范围 `[left, mid)`，右子树范围 `[mid + 1, right)`

## 递归：左闭右闭 [left, right]

```java
class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        TreeNode root = traversal(nums, 0, nums.length - 1);
        return root;
    }

    // 左闭右闭区间[left, right]
    private TreeNode traversal(int[] nums, int left, int right) {
        if (left > right) return null;

        int mid = left + ((right - left) >> 1);
        TreeNode root = new TreeNode(nums[mid]);
        root.left = traversal(nums, left, mid - 1);
        root.right = traversal(nums, mid + 1, right);
        return root;
    }
}
```

**关键点：**
- 初始调用传 `nums.length - 1`，因为右边界是闭区间
- 终止条件是 `left > right`（区间为空）
- 左子树范围 `[left, mid - 1]`，右子树范围 `[mid + 1, right]`
- `>> 1` 等价于 `/2`，位运算写法

## 迭代：左闭右闭 [left, right]

用**三个队列**模拟递归过程：分别存储待处理的节点、左边界、右边界。

```java
class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        if (nums.length == 0) return null;

        // 根节点初始化
        TreeNode root = new TreeNode(-1);
        Queue<TreeNode> nodeQueue = new LinkedList<>();
        Queue<Integer> leftQueue = new LinkedList<>();
        Queue<Integer> rightQueue = new LinkedList<>();

        // 根节点入队列
        nodeQueue.offer(root);
        // 0为左区间下标初始位置
        leftQueue.offer(0);
        // nums.length - 1为右区间下标初始位置
        rightQueue.offer(nums.length - 1);

        while (!nodeQueue.isEmpty()) {
            TreeNode currNode = nodeQueue.poll();
            int left = leftQueue.poll();
            int right = rightQueue.poll();
            int mid = left + ((right - left) >> 1);

            // 将mid对应的元素给中间节点
            currNode.val = nums[mid];

            // 处理左区间
            if (left <= mid - 1) {
                currNode.left = new TreeNode(-1);
                nodeQueue.offer(currNode.left);
                leftQueue.offer(left);
                rightQueue.offer(mid - 1);
            }

            // 处理右区间
            if (right >= mid + 1) {
                currNode.right = new TreeNode(-1);
                nodeQueue.offer(currNode.right);
                leftQueue.offer(mid + 1);
                rightQueue.offer(right);
            }
        }
        return root;
    }
}
```

**关键点：**
- 先创建值为 `-1` 的占位节点入队，之后在循环中赋真实值 `nums[mid]`
- 三个队列保持同步：`nodeQueue` 存节点，`leftQueue`/`rightQueue` 存该节点对应的区间边界
- 判断 `left <= mid - 1` 和 `right >= mid + 1` 来决定是否需要创建左/右子节点

## 踩坑记录

1. **递归中用 `nums.length` 算 mid** — `nums.length` 是固定值，每层递归算出的 mid 都一样，导致无限递归栈溢出。必须用 `left` 和 `right` 来计算 mid
2. **递归参数写死为 `0` 和 `nums.length - 1`** — 和上面同理，递归体内必须用参数 `left`/`right` 来缩小范围，否则永远在处理整个数组
3. **区间风格不统一** — 左闭右开和左闭右闭的终止条件、边界传递、初始调用都不同，混用会导致漏元素或越界

## 复杂度分析

- **时间复杂度：O(n)** — 每个元素恰好访问一次
- **空间复杂度：O(log n)** — 递归栈深度 / 队列最大容量为树的高度

## 总结

1. 本题本质是**二分法构建树**：有序数组取中点 → 根节点，左半段 → 左子树，右半段 → 右子树
2. 递归写法需要明确**区间定义**（左闭右开 vs 左闭右闭），终止条件和边界传递必须与区间风格一致
3. 迭代写法用**三个队列**模拟递归的"待处理栈帧"，思路是先建占位节点再赋值
4. `mid = left + ((right - left) >> 1)` 防止整数溢出，是标准写法
